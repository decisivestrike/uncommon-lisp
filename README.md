# Uncommon Lisp

Uncommon Lisp основан на выражениях. Выражение это список в круглых скобках, первым элементом которого является функция.

## Datatypes

- Number
- String
- Bool
- Nil
- List

## Examples

```py
(var name "Hello")
(var age (add 10 10))
(var list [1 2 3])

# Empty expression
() -> Nil

# Простая функция сложения двух чисел
(func addTwoNumbers [a b])
  (add a b))

(addTwoNumbers 5 3)
```
